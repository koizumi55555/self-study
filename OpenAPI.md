OpenAPI

  - OpenAPI Specificationは、WebAPI（REST）のインターフェースを記述するための仕様。フォーマット。
  - OpenAPI Specification(OAS、OpenAPI Spec)
  - 人間とコンピュータの両方が読める
  - 言語に依存しない
- 記述ファイル

  - JSON
  - YAML　※相互変換が可能
- 利用方法

  - APIドキュメントとして
  - APIツールとして

    - API ドキュメントを生成
    - API モックサーバーを立てられる
    - クライアントコード、サーバサイドコードの自動生成
